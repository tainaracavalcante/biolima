Lista de rotina de estudo

  1. Organizar o material e definir as metas do dia (5–10 min).
  2. Revisar o conteúdo estudado no dia anterior (20 min).
  3. Estudar a primeira matéria com foco total (50 min).
  4. Fazer uma pausa curta para descanso (10 min).
  5. Estudar a segunda matéria com anotações e resumos (50 min).
  6. Resolver exercícios sobre as duas matérias (40 min).
  7. Corrigir os erros e anotar as dúvidas (20 min).
  8. Assistir a uma aula complementar ou vídeo explicativo (30 min).
  9. Fazer uma revisão rápida com flashcards ou mapas mentais (20 min).
  10. Planejar o cronograma de estudos do dia seguinte (10 min).

Lista de passos para criar um HTML

  1. Abrir um editor de texto (como VS Code ou Bloco de Notas).
  2. Criar um novo arquivo e salvar com a extensão .html (ex: index.html).
  3. Escrever a estrutura básica com !DOCTYPE html>.
  4. Adicionar a tag html para iniciar o documento.
  5. Criar a seção head com title para o título da página.
  6. Adicionar títulos usando h1 ou h2.
  7. Abrir a tag body para inserir o conteúdo visível.
  8. Adicionar títulos usando h1> ou h2>.
  9. Inserir parágrafos com p> e outros elementos necessários.
  10. Salvar o arquivo após finalizar as edições.
  11. Abrir o arquivo no navegador para visualizar o resultado.

Lista de um checklist para entregar uma atividade

  1. Conferir se todas as perguntas da atividade foram respondidas.
  2. Verificar se o conteúdo está completo e bem explicado.
  3. Revisar ortografia e gramática.
  4. Conferir se as normas solicitadas (formatação, fonte, tamanho) foram seguidas.
  5. Checar se o nome, data e identificação estão corretos.
  6. Confirmar se todas as páginas estão numeradas (se necessário).
  7. Anexar arquivos ou materiais complementares solicitados.
  8. Garantir que o trabalho esteja salvo no formato correto (PDF, Word, etc.).
  9. Revisar novamente para corrigir possíveis erros finais.
  10. Enviar ou entregar dentro do prazo estabelecido.
